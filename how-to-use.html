How to Use?

How to Use the Attendance Calculator?

1. Set Your Attendance Goal: Enter the attendance percentage you want (e.g., 75%, 80%, up to 100%).

2. Enter Total Classes: Input the total number of classes held.

3. Enter Classes Attended: Input the number of classes you've attended.

4. Calculate: Click Calculate to see your current attendance status.

How to Use the GPA Calculator?

1. Select Your Grade: Use the first dropdown to choose the grade you received for each subject (e.g., S, A, B, etc.). Each grade has a specific numeric value (S = 10, A = 9, B = 8, etc.).
